CACHE SIMULATOR


Project Mission:

Create a program that can simulate up to two caches of E20 assembly programs

Focused on the concept of caching




The cache simulator is a program that has been tested and verified with many different cases

What does it do?

The cache simulator is to be used after using the assembler, so its input is a program in machine code. Then it reads through the instructions, executing each line of the program and updating any registers and values that have been modified. The program also simulates caching-- it can build up to two caches to simulate a program and it also maintains the caches which includes various algorithms for the different types of caches
